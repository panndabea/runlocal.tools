Strip image metadata.

Remove hidden metadata from images directly in your browser. Useful before sharing photos, screenshots, product images, or sensitive visual material online.

Privacy first Images are decoded and re-exported locally through the browser canvas. The exported file contains only the rendered pixels, without EXIF, GPS, camera, or author metadata.
